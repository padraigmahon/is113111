About Cloud Services:

A cloud service is any service made available to users on demand via the Internet from a cloud computing
provider's servers as opposed to being provided from a company's own on-premises servers.

Cloud Services are much more accessible and applicable to users
than a standard offline service as it can be utilized anywhere anytime using any device with an internet connection.

Advantages to cloud services:


  1. Drive down cost.

  2. Improve accessiblity.

  3. Avoid large capital expenditure on hardware and upgrades.
Examples of Cloud Services:
  1. Amazon Web Services

  2. Microsoft Azure

  3. Google Cloud Platform

  4. Adobe

  5. VMware

  6. IBM Cloud

  7. Rackspace
